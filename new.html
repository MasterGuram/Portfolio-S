Portfolio Template Setup Guide

Welcome to the Portfolio template by MasterGuram. This guide will walk you through the steps to install, customize, and update your personal portfolio website using this template.

1. Installation

  1. Unzip downloaded ZIP file
  2. Open the folder with any code editor (e.g., VS Code).
  3. You can preview the site by opening index.html in your browser.

2. Editing the Template Files

Open the project folder in your preferred code editor (e.g., VS Code).

  • index.html: Main HTML file containing the structure of your portfolio.
  • css/: Contains CSS files for styling.
  • js/: Contains JavaScript files for interactivity.
  • img/: Place your images here.
  • webfonts/: Contains font files used in the template.
  • CV.pdf: Replace with your own CV or resume.

3. Updating the Template

After making changes, view them by opening index.html in your browser.

To deploy your portfolio online:

  1. Create a GitHub repository.
  2. Push your local files to it using Git.
  3. In your repository settings, go to Pages and enable GitHub Pages from the root of the main branch.
  4. Your site will be live at: https://yourusername.github.io/your-repo-name/

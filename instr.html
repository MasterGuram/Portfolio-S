📘 Portfolio Template - Setup & Customization Guide

🛠 Installation

  1. Unzip downloaded ZIP file
  2. Open the folder with any code editor (e.g., VS Code).
  3. You can preview the site by opening index.html in your browser.

📝 Editing Files

Update personal info, text, and images directly in these files:

  • index.html: main layout, personal intro, project section
  • img/: replace profile and project images
  • css/style.css: edit colors, fonts, layout spacing
  • js/main.js: adjust interactive behavior (e.g., navigation effects)

🔄 Editing index.html Content

  1. In index.html, locate the projects section.
  2. Duplicate and modify an existing project block: 
    <div class="project">
  <img src="img/myproject.jpg" alt="My Project">
  <h3>Project Name</h3>
  <p>Short description...</p>
  <a href="https://github.com/yourproject">View Code</a>
</div>
  3. Save and refresh the browser to see changes.

🌐 Optional: Deploy to GitHub Pages

  1. Go to your GitHub repo → Settings → Pages
  2. Set the source to main branch, root folder
  3. Wait a few minutes, then access your site at https://yourusername.github.io/Portfolio-S

Feel free to customize and expand this project to reflect your own style and work!
